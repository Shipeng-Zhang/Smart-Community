# 启动说明

推荐入口：

```bash
cd /mnt/e/iot
python3 start_community_system.py --port 8080
```

打开浏览器：

```text
http://127.0.0.1:8080/login.html
```

登录后进入主系统：

```text
http://127.0.0.1:8080/index.html
```

演示账号：

- 账号：`admin@community.local`
- 密码：`Admin@12345`

页面包含：

- 登录 / 注册
- 总览
- 工单
- 告警
- 设备
- 创新
- 设置
